Quick start guide

1. Install angular-funky-button via Bower

The recommended way to install angular-funky-button is to use the Bower package: 

bower install --save angular-funky-button

2. Include js/css files

Include the angular-funky-button files:

  • required: angular-funky-button.js - the directive itself
  • optional: angular-funky-button.css - some common stylings
...
<head>
    ...
    <link rel="stylesheet" href="bower_components/angular-funky-button/dist/angular-funky-button.css">
    <script type="text/javascript" src="bower_components/angular-funky-button/dist/angular-funky-button.min.js"></script>
    ...
</head>
...

3. Your first button



<button funky-button fubu-click="angular.noop()">
    <span fubu-default>click me</span>
    <span fubu-confirm>are you sure?</span>
    <span fubu-success>done!</span>
</button>
