Button with confirmation

  • Asks the user for a confirmation
  • Shows a working and success message
  • The working promise resolves to true
Code
<button class="btn" funky-button fubu-click="work()">
    <span fubu-default fubu-class="btn-primary">
        <i class="fa fa-fw fa-envelope-o"></i> send mail
    </span>
    <span fubu-confirm fubu-class="btn-primary">
        <i class="fa fa-fw fa-question"></i> are you sure?
    </span>
    <span fubu-working fubu-class="btn-primary">
        <i class="fa fa-fw fa-spin fa-circle-o-notch"></i> sending mail
    </span>
    <span fubu-success fubu-class="btn-success">
        <i class="fa fa-fw fa-check"></i> mail sent
    </span>
</button>

Button with error - click to continue

  • The action fails → displays an error message
  • The user must click the error message to continue
Code

Set fubu-dismiss="0" on the fubu-error element to disable the time limit for error messages. 

<button class="btn" funky-button fubu-click="workError()">
    <span fubu-default fubu-class="btn-primary">
        <i class="fa fa-fw fa-envelope-o"></i> send mail
    </span>
    <span fubu-working fubu-class="btn-primary">
        <i class="fa fa-fw fa-spin fa-circle-o-notch"></i> sending mail
    </span>
    <span fubu-success fubu-class="btn-success">
        <i class="fa fa-fw fa-check"></i> mail sent
    </span>
    <span fubu-error fubu-dismiss="0" fubu-class="btn-danger">
        <i class="fa fa-fw fa-exclamation-triangle"></i> can't send mail
    </span>
</button>

Button with error - programmatically dismiss error

  • The action fails → displays an error message
  • The error will be dismissed by changing a flag

Code

In this example the error message is dismissed when "dismissError" changes to true. Set fubu-dismiss="dismissError" on the fubu-error element. 

<button class="btn" funky-button fubu-click="workError()">
    <span fubu-default fubu-class="btn-primary">
        <i class="fa fa-fw fa-envelope-o"></i> send mail
    </span>
    <span fubu-working fubu-class="btn-primary">
        <i class="fa fa-fw fa-spin fa-circle-o-notch"></i> sending mail
    </span>
    <span fubu-success fubu-class="btn-success">
        <i class="fa fa-fw fa-check"></i> mail sent
    </span>
    <span fubu-error fubu-dismiss="dismissError" fubu-class="btn-danger">
        <i class="fa fa-fw fa-exclamation-triangle"></i> can't send mail
    </span>
</button>
